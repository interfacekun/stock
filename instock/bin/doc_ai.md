# bin Root Folder

## Description
This is the executable scripts and startup utilities directory containing various command-line tools and startup scripts for the stock trading platform. The bin folder houses shell scripts and batch files that facilitate application deployment and operation.

## Contents Overview
- restart_web.sh: Script to restart web services
- run_cron.sh: Script to execute cron jobs
- run_job.bat: Batch script for running jobs on Windows
- run_job.sh: Shell script for running jobs on Unix systems
- run_trade.bat: Batch script for running trading services on Windows
- run_web.bat: Batch script for running web services on Windows
- run_web.sh: Shell script for running web services on Unix systems

## Key Components
- `run_job.sh`: Main script for executing application jobs
- `run_web.sh`: Script for starting web services
- `restart_web.sh`: Script for restarting web services
