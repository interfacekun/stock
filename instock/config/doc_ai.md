# config Root Folder

## Description
This is the configuration files and settings directory containing various configuration parameters and environment-specific settings for the stock trading platform. The config folder houses essential files that define application behavior and external service connections.

## Contents Overview
- .gitignore: Git ignore configuration
- eastmoney_cookie.txt: Cookie storage for East Money integration
- proxy.txt: Proxy server configuration file
- trade_client.json: Trading client configuration parameters

## Key Components
- `trade_client.json`: Configuration for trading client connections
- `eastmoney_cookie.txt`: Authentication data for East Money API
- `proxy.txt`: Network proxy settings
