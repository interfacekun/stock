# cron.workdayly Root Folder

## Description
This is the workday scheduled job definitions directory containing tasks that run on workdays for the stock trading platform. The cron.workdayly folder houses job scripts and configurations that execute on business days to perform trading-related operations and market data updates.

- Directory containing workday scheduled job definitions
